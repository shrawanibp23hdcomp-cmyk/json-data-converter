
# JSON Data Converter Project

## Overview
This project converts telemetry data from two different JSON formats into a unified structure.

## Features
- Handles multiple input formats
- Converts timestamps to milliseconds
- Extracts and restructures nested data
- Includes unit testing for validation

## Technologies Used
- Python
- JSON
- unittest module

## How to Run
1. Download the files
2. Run:
   python main.py

## Output
All tests pass successfully, ensuring correct data transformation.
